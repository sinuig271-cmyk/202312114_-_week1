## 1. 제품 개요

### 1.1 제품명

**ChaosLab** — 이중 진자 카오스 시뮬레이터

### 1.2 한 줄 요약

물리 학습자가 이중 진자의 카오스 특성을 직접 탐구할 수 있는 인터랙티브 데스크탑 시뮬레이션 앱.

### 1.3 배경 및 목적

카오스 이론은 대학 물리학 커리큘럼에서 중요하게 다뤄지지만, 수식만으로는 직관적 이해가 어렵다. 이중 진자는 결정론적 시스템이면서도 초기 조건에 극도로 민감한 카오스 거동을 보이는 대표적 사례다. 이 앱은 애니메이션, 나비효과 비교, 로렌츠 어트랙터, 위상공간 시각화를 통해 추상적인 카오스 개념을 체감 가능한 형태로 제공한다.

### 1.4 목표 사용자

| 구분 | 설명 |
|------|------|
| 주 사용자 | 대학교 물리학과 학부생 (1~3학년) |
| 부 사용자 | 고등학생 물리 심화 학습자, 물리 교육자 |
| 전제 지식 | 뉴턴 역학 기초, 미적분 개념 |

## 2. 핵심 기능 (MVP 범위)

### 2.1 탭 구조 개요

```
ChaosLab
├── Tab 1: 이중 진자 시뮬레이션  (실시간 애니메이션)
├── Tab 2: 나비효과              (초기 조건 민감성 비교)
├── Tab 3: 로렌츠 어트랙터       (독립 3D 시각화)

### 2.2 Tab 1 — 이중 진자 시뮬레이션

**목적:** 이중 진자의 실시간 운동을 시각화하고 파라미터를 직접 조작한다.

#### 기능 요구사항

| ID | 기능 | 설명 |
|----|------|------|
| F1-01 | 실시간 애니메이션 | Runge-Kutta 4차(RK4) 수치 적분으로 운동방정식 풀이, 60fps 목표 |
| F1-02 | 파라미터 슬라이더 | 질량(m1, m2), 막대 길이(L1, L2), 초기 각도(θ1, θ2), 중력가속도(g) 조절 |
| F1-03 | 궤적 잔상 | 끝점(추 2)의 궤적을 색상 페이드로 표시 (최근 N 포인트) |
| F1-04 | 재생 제어 | 시작 / 일시정지 / 초기화 버튼 |
| F1-05 | 에너지 표시 | 현재 운동에너지 · 위치에너지 · 전체 에너지를 숫자로 실시간 표시 |

#### 운동방정식

RK4로 풀 연립 ODE (라그랑지안 유도):

```
θ1'' = f1(θ1, θ2, ω1, ω2, m1, m2, L1, L2, g)
θ2'' = f2(θ1, θ2, ω1, ω2, m1, m2, L1, L2, g)
```

#### UI 레이아웃

```
┌─────────────────────────────────────────────┐
│  [캔버스: 진자 애니메이션]      │ 파라미터  │
│                                 │ m1 ──●──  │
│          ●──                    │ m2 ──●──  │
│             ──●                 │ L1 ──●──  │
│           (궤적 잔상)           │ L2 ──●──  │
│                                 │ θ1 ──●──  │
│                                 │ θ2 ──●──  │
├─────────────────────────────────┤ g  ──●──  │
│  KE: 0.00 J  PE: 0.00 J  E: 0.00 J         │
│  [▶ 시작]  [⏸ 일시정지]  [↺ 초기화]        │
└─────────────────────────────────────────────┘
```

### 2.3 Tab 2 — 나비효과

**목적:** 초기 조건의 미세한 차이가 시간이 지남에 따라 얼마나 발산하는지 체감한다.

#### 기능 요구사항

| ID | 기능 | 설명 |
|----|------|------|
| F2-01 | 다중 진자 동시 시뮬레이션 | 초기 각도에 미세 차이(Δθ)를 둔 N개(기본값 5) 진자를 동시 렌더링 |
| F2-02 | 색상 구분 | 각 진자마다 고유 색상 부여 (Plasma 또는 Viridis 팔레트) |
| F2-03 | 편차 Δθ 설정 | Δθ를 슬라이더로 조절 (범위: 0.001° ~ 1.0°) |
| F2-04 | 발산 거리 그래프 | 시간 축 대비 끝점 간 평균 거리 그래프를 하단에 실시간 표시 |
| F2-05 | 리아푸노프 지수 추정 | 발산 속도로부터 리아푸노프 지수(λ) 근사값 계산 후 표시 |

#### UI 레이아웃

```
┌──────────────────────────────────────────────┐
│  [캔버스: N개 진자 동시 표시]                │
│   🔴🟠🟡🟢🔵 (색상별 궤적 잔상)             │
├──────────────────────────────────────────────┤
│  [발산 거리 그래프 — 시간 vs 거리]           │
│  λ ≈ 0.42 s⁻¹   Δθ: 0.01°  N: 5            │
│  [▶ 시작]  [↺ 초기화]                        │
└──────────────────────────────────────────────┘
```

### 2.4 Tab 3 — 로렌츠 어트랙터

**목적:** 이중 진자와 별개로, 대표적인 카오스 어트랙터인 로렌츠 시스템을 3D로 시각화한다.

#### 기능 요구사항

| ID | 기능 | 설명 |
|----|------|------|
| F3-01 | 3D 궤적 렌더링 | matplotlib Axes3D 또는 PyQtGraph로 로렌츠 방정식 궤적 표시 |
| F3-02 | 파라미터 조절 | σ (sigma), ρ (rho), β (beta) 슬라이더 |
| F3-03 | 마우스 회전 | 드래그로 3D 뷰 회전 |
| F3-04 | 다중 궤적 비교 | 초기 조건이 미세하게 다른 2개 궤적을 서로 다른 색으로 표시 |
| F3-05 | 애니메이션 속도 | 궤적 생성 속도 조절 슬라이더 |

#### 로렌츠 방정식

```
dx/dt = σ(y − x)
dy/dt = x(ρ − z) − y
dz/dt = xy − βz

기본값: σ=10, ρ=28, β=8/3
```

#### UI 레이아웃

```
┌──────────────────────────────────────────────┐
│  [3D 캔버스 — 로렌츠 어트랙터 궤적]          │
│         (마우스 드래그로 회전)               │
├──────────────────────────────────────────────┤
│  σ ──●──  ρ ──●──  β ──●──  속도 ──●──      │
│  [▶ 시작]  [⏸ 일시정지]  [↺ 초기화]        │
└──────────────────────────────────────────────┘
```

### 2.5 Tab 4 — 위상공간

**목적:** 이중 진자의 상태를 위상공간(θ-ω 평면)에 표시해 카오스 궤도의 비주기성을 확인한다.

#### 기능 요구사항

| ID | 기능 | 설명 |
|----|------|------|
| F4-01 | 위상 궤적 실시간 플롯 | Tab 1 시뮬레이션 데이터를 가져와 (θ1, ω1), (θ2, ω2) 위상 궤적 표시 |
| F4-02 | 이중 패널 | 진자 1과 진자 2의 위상 궤적을 나란히 표시 |
| F4-03 | 포앙카레 단면 | θ2=0 조건 만족 시점의 (θ1, ω1) 점을 별도 마킹 |
| F4-04 | 데이터 내보내기 | 위상 궤적 데이터를 CSV로 저장 |
| F4-05 | 시간 범위 선택 | 표시할 시간 구간 슬라이더로 선택 |

#### UI 레이아웃

```
┌────────────────────┬────────────────────┐
│  θ1 - ω1 위상공간  │  θ2 - ω2 위상공간  │
│   (포앙카레 단면   │                    │
│    점 ● 표시)      │                    │
├────────────────────┴────────────────────┤
│  시간 구간: [──────●──────────]         │
│  [📥 CSV 내보내기]                      │
└─────────────────────────────────────────┘
```

## 3. 기술 스택

| 구분 | 선택 | 비고 |
|------|------|------|
| UI 프레임워크 | PySide6 | Qt 6 기반, 탭 위젯 |
| 수치 계산 | NumPy, SciPy | RK4 적분 |
| 2D 시각화 | Matplotlib (Qt 백엔드) | 위상공간, 발산 그래프 |
| 3D 시각화 | Matplotlib Axes3D 또는 PyQtGraph | 로렌츠 어트랙터 |
| 애니메이션 루프 | QTimer (16ms, ~60fps) | 메인 루프 |
| 데이터 저장 | CSV (csv 표준 라이브러리) | 위상공간 내보내기 |
| Python 버전 | 3.11+ | |

### 3.1 프로젝트 구조

```
chaoslab/
├── main.py                  # 앱 진입점

│   └── tab_phase.py         # Tab 4
├── physics/
│   ├── double_pendulum.py   # 운동방정식 + RK4
│   └── lorenz.py            # 로렌츠 방정식
