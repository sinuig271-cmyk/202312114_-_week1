## 1. 개요 (Overview)

### 1.1 제품 목적

머신러닝 입문자 및 학생이 딥러닝의 핵심 개념인 **Activation Function(활성화 함수)**을 직관적으로 이해할 수 있도록, 함수 그래프 시각화 · Gradient 표시 · 수식 설명을 제공하는 데스크탑 GUI 애플리케이션이다.

### 1.2 배경 및 문제 정의

| 문제 | 설명 |
|------|------|
| 추상적 개념 | Activation Function은 수식만으로는 직관적 이해가 어렵다 |
| 파라미터 영향 불명확 | alpha, beta 등 하이퍼파라미터가 함수 형태에 미치는 영향을 체감하기 힘들다 |
| Gradient 연결 부족 | 역전파(Backpropagation)와의 연결 고리인 미분값을 함께 학습하기 어렵다 |
| 분산된 자료 | 각 함수별 수식·특징·사용 맥락이 한 곳에 정리되어 있지 않다 |

### 1.3 제품 비전

> "파라미터 슬라이더를 움직이는 순간, 함수가 살아있는 것처럼 반응한다 — 수식, 그래프, Gradient를 한 화면에서."

## 3. 기능 요구사항 (Functional Requirements)

### 3.1 핵심 기능 (Core Features)

#### F-01. Activation Function 목록 및 선택

- 사이드바에 지원 함수 전체 목록을 표시한다
- 사용자가 함수를 클릭하면 즉시 메인 영역에 반영된다
- 기본·고급 카테고리로 그룹핑한다

**지원 함수 목록**

| 카테고리 | 함수 |
|----------|------|
| 기본 | ReLU, Sigmoid, Tanh, Softmax |
| 고급 | Leaky ReLU, ELU, GELU, Swish, PReLU, SELU |

#### F-02. 함수 그래프 시각화

- Matplotlib 기반 그래프를 PySide6 캔버스에 임베드하여 표시한다
- x축 범위는 기본 [-5, 5]이며 사용자가 조절 가능하다
- 함수 곡선과 Gradient(미분) 곡선을 동일 그래프에 겹쳐 표시한다
- 함수 선·Gradient 선은 색상으로 구분하며 범례(Legend)를 제공한다

#### F-03. 실시간 파라미터 조절

- 선택한 함수의 하이퍼파라미터(alpha, beta 등)를 슬라이더 및 스핀박스로 조절한다
- 파라미터 변경 시 그래프가 실시간으로 업데이트된다

**파라미터 예시**

| 함수 | 조절 파라미터 |
|------|---------------|
| Leaky ReLU | alpha (기울기, 0.0 ~ 1.0) |
| ELU | alpha (스케일, 0.1 ~ 5.0) |
| GELU | 없음 (고정) |
| Swish | beta (0.1 ~ 5.0) |

#### F-04. 수식 및 수학적 설명 패널

- 현재 선택된 함수의 수식을 LaTeX 형식으로 렌더링하여 표시한다
- Gradient(미분식)도 함께 표시한다
- 함수의 특징, 장단점, 주요 사용 맥락(Output layer / Hidden layer 등)을 텍스트로 설명한다

#### F-05. 한/영 언어 전환

- 상단 툴바에 언어 전환 버튼(KO / EN)을 제공한다
- 전환 시 UI 레이블, 설명 텍스트, 파라미터 라벨이 즉시 변경된다
- 수식은 언어와 무관하게 동일하게 표시된다

### 5.1 레이아웃 구조

```
┌─────────────────────────────────────────────────────┐
│  [툴바]  앱 제목           언어: KO / EN             │
├──────────────┬──────────────────────────────────────┤
│  [사이드바]   │  [메인 영역]                          │
```

- **즉각성**: 모든 인터랙션(함수 선택, 파라미터 변경)은 즉시 반응
- **명료성**: 입문자 기준으로 용어 설명 포함, 전문 용어 사용 최소화
- **집중**: 한 화면에서 그래프 · 수식 · 설명을 모두 확인 가능

## 6. 지원 함수 상세 명세

| 함수 | 수식 | 주요 특징 |
|------|------|-----------|
| ReLU | `max(0, x)` | 가장 널리 사용, Dying ReLU 문제 |
| Sigmoid | `1 / (1 + e^{-x})` | 이진 분류 출력층, Vanishing Gradient |
| Tanh | `(e^x - e^{-x}) / (e^x + e^{-x})` | [-1,1] 범위, 중심화된 출력 |
| Softmax | `e^{xi} / Σe^{xj}` | 다중 분류 출력층 |
| Leaky ReLU | `max(αx, x)` | Dying ReLU 개선 |
| ELU | `x if x>0 else α(e^x - 1)` | 음수 출력 허용 |
| GELU | `x · Φ(x)` | Transformer 계열에서 주로 사용 |
| Swish | `x · σ(βx)` | Self-gated, 부드러운 곡선 |
| PReLU | `max(αx, x)` (α 학습) | Leaky ReLU의 학습 가능 버전 |
| SELU | `λ · ELU(x, α)` | Self-normalizing |

## 7. 제약사항 및 가정

- 이 애플리케이션은 **독립 실행형 데스크탑 앱**이며, 웹/모바일 지원은 범위 외
- Softmax는 다변수 함수이므로 단일 입력 시각화 시 근사 표현 처리
- 사용자 정의 함수 기능(F-09)은 v1.0 범위 외, v2.0 고려 사항
- LaTeX 렌더링은 Matplotlib의 내장 `mathtext` 또는 외부 라이브러리 활용
