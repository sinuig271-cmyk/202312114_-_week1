## 2. 배경 및 목적 (Background & Objectives)

### 배경

퍼셉트론은 딥러닝의 근간이 되는 핵심 개념이지만, 가중치·편향·활성화 함수·학습 규칙의 관계를 텍스트와 수식만으로 이해하기 어렵다. 특히 결정 경계가 어떻게 움직이는지, 학습이 왜 수렴하는지를 시각적으로 체험할 수 없는 것이 학습 장벽으로 작용한다.

### 목적

- 퍼셉트론의 **구조(입력·가중치·편향·활성화·출력)**를 다이어그램으로 직관적으로 전달
- **슬라이더**로 가중치/편향을 실시간 조작하며 출력 변화 체험
- AND / OR 게이트 학습 **애니메이션**으로 퍼셉트론 학습 알고리즘 시각화
- **결정 경계** 그래프로 선형 분리 가능성 개념 체험 (XOR 한계 포함)
- 퀴즈를 통한 자가 점검 및 즉각 피드백 제공

```

```

### Tab 1 — 개요 (Overview)

- 퍼셉트론이란? — Frank Rosenblatt(1958) 역사적 배경
- 핵심 수식: `y = f(w₁x₁ + w₂x₂ + b)` 표시
- 생물학적 뉴런과 퍼셉트론 구성 요소 비교표
- 퍼셉트론의 한계(XOR 문제) 및 MLP 필요성 안내
- 이 앱의 학습 경로 안내

### Tab 2 — 구조 시각화 (Structure)

- QPainter 기반 퍼셉트론 뉴런 다이어그램
  - 입력 노드(x₁, x₂), 편향 노드(+1)
  - 가중치 엣지 (두께·색상으로 값 표현)
  - 가중합 노드(Σ), 활성화 함수 노드(f), 출력 노드(y)
- 가중치(w₁, w₂) 슬라이더 (범위: −2.0 ~ +2.0)
- 편향(b) 슬라이더 (범위: −2.0 ~ +2.0)
- 입력값(x₁, x₂) 버튼 (0 / 1 토글)
- 계산 과정 패널: z 값, 활성화, 출력 y 실시간 표시

### Tab 3 — 학습 애니메이션 (Learning)

- AND / OR 게이트 콤보박스 선택
- 학습률(η) 슬라이더 (0.01 ~ 0.50)
- 애니메이션 속도 슬라이더 (50ms ~ 1000ms)
- [▶ 시작] / [⏭ 1스텝] / [⏸ 일시정지] / [↺ 초기화] 버튼
- 에포크별 가중치·오류 수 상태 표시
- Matplotlib 오류율 그래프 (에포크 vs 오류 수)
- 학습 로그 텍스트 패널 (스텝별 상세 기록)

### Tab 4 — 결정 경계 (Decision Boundary)

- AND / OR / XOR 게이트 라디오 버튼 선택
- Matplotlib 2D 산점도: 데이터 포인트 (0→빨간색, 1→초록색)
- 가중치/편향에 따른 결정 경계선 실시간 표시
- 가중치(w₁, w₂, b) 슬라이더 (이 탭 독립)
- XOR 선택 시 "선형 분리 불가능" 경고 패널 + MLP 필요성 설명
- 분리 가능 여부 상태 표시

### Tab 5 — 퀴즈 (Quiz)

- 5문제, 중간 난이도
- 유형: 객관식 4지선다 위주
- 필수 주제: 편향 역할, 수치 계산, 학습 규칙, XOR 한계, 수렴 정리
- 문제별 즉각 피드백 (정오답 + 해설 텍스트)
- 최종 결과: 점수(N/5), 퍼센트, 등급(A/B/C/F), 재도전 버튼

## 5. 핵심 기능 명세 (Feature Specifications)

### F-1: 퍼셉트론 구조 다이어그램

| 속성 | 내용 |
|------|------|
| 렌더링 방식 | PySide6 QPainter (QWidget 상속) |
| 입력 노드 | x₁, x₂, bias(+1) — 원형 노드 |
| 엣지 표현 | 두께 = \|w\| 비례, 색상 = 양수(파란색) / 음수(빨간색) |
| 가중치 레이블 | 엣지 중간에 wᵢ 값 표시 |
| 업데이트 | 슬라이더 valueChanged 시 즉시 repaint() |

### F-2: 퍼셉트론 학습 애니메이션

| 속성 | 내용 |
|------|------|
| 알고리즘 | Rosenblatt 학습 규칙: `w ← w + η(y−ŷ)x`, `b ← b + η(y−ŷ)` |
| 지원 게이트 | AND, OR |
| 최대 에포크 | 50 |
| 타이머 | QTimer (50ms ~ 1000ms 간격 조절) |
| 수렴 감지 | 에포크 오류 수 = 0 시 자동 정지 + 완료 메시지 |

### F-3: 결정 경계 시각화

| 속성 | 내용 |
|------|------|
| 렌더링 방식 | Matplotlib (FigureCanvasQTAgg 임베드) |
| 지원 게이트 | AND, OR, XOR |
| 경계선 공식 | w₁x + w₂y + b = 0 |
| XOR 처리 | 경고 패널 표시 + 학습 불가 안내 |
| 업데이트 | 슬라이더 변경 시 즉시 canvas.draw() |

### F-4: 퀴즈

| 속성 | 내용 |
|------|------|
| 문제 수 | 5문제 |
| 난이도 | 중간 |
| 주제 | 편향 역할 / 수치 계산 / 학습 규칙 / XOR 한계 / 수렴 정리 |
| 피드백 | 문제 제출 후 즉각 정오답 + 해설 표시 |
| 결과 화면 | 총점, 퍼센트, 등급, 재도전 버튼 |
